In posts moving forward I am going to be sharing a lot of things I have put together using LLMs. Part of this too is me looking for solutions to tasks and problems I have, but the solutions need to run without a server (cause who wants a server. These aren't robust solutions made for everyone, but since this post is retrospective (I wrote it mid May 2026), I can say there is lots of room to push into this area.

&nbsp;
KISS - Keep it simple stupid. Yep, that's the idea. Maybe it is a pretty big static html that handles memory via local json or some shared google sheet... but it works? Cool.

&nbsp;
Houseplant Programming - see [this](https://hannahilea.com/blog/houseplant-programming/) blog post. Basically it is the idea that we can make things and they don't have to be full production level, or maybe they really only need to work for me and my specific use case. And that's fine.
