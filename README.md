This is my personal blog on science, specifically proteomics topics and conversations originating on Twitter. The blog is hosted on GitHub Pages and based on the beautifully simple [Jekyll](https://jekyllrb.com/)-powered theme [Reverie](https://github.com/amitmerchant1990/reverie) by [Amit Merchant](https://github.com/amitmerchant1990). It has since been modified to use [giscus](https://giscus.app/) and a custom built compose UI (available within [Hyde](github.com/neely/hyde)).
