This is my [personal blog](https://neely.github.io) on science, specifically proteomics topics and conversations (originally was originating on sciTwitter TeamMasssSpec before that died). The blog is hosted on GitHub Pages and based on the beautifully simple [Jekyll](https://jekyllrb.com/)-powered theme [Reverie](https://github.com/amitmerchant1990/reverie) by [Amit Merchant](https://github.com/amitmerchant1990). It has since been modified to use [giscus](https://giscus.app/) and a custom built compose UI (available within [Hyde](https://github.com/neely/hyde)).
